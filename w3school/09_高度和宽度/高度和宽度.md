# css 高度和宽度

#### 设置高度和宽度
height属性用于设置高度。
width属性用于设置宽度。

<font color="red">注意：height和width不包括 内边距、边框和外边距</font>

#### 高度和宽度的值
height和width属性可设置的值如下：
- auto，默认，由浏览器计算
- length，指定单位的长度
- %，以包含块的百分比定义值
- inherit，从其父元素的值集成宽高值

#### 设置max-width
max-width属性用于设置元素的最大宽度。
默认值为none，意味着没有最大宽度。

如果浏览器窗口小于元素的宽度时，浏览器会将水平滚动条添加到页面。
